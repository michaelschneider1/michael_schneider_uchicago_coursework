# CMSC 14100 (Winter 2025) Coursework

**Language Used:** Python

**Concepts Covered**
1. Python Fundamentals: Variables, conditionals, iteration, and core computational mechanisms for solving problems.
2. Data Structures: Working with Python's built-in structures (lists, strings, tuples, and dictionaries) to represent and manipulate data.
3. Functions & AbstractionL Designing and calling functions, using programming abstractions to reduce complexity and improve code reusability.
4. Classes & Objects: Introduction to object-oriented programming, including designing and using classes to model real-world problems.
5. Recursion: Writing recursive definitions for both data and functions, and building and traversing recursive data structures.
6. Software Development Tools: Practical experience with Linux command-line, Git version control, and VSCode.

Final Grade: A
