# CMSC 14200 (Spring 2025)

**Language Used:** Python

**Concepts Covered**
Part 1: Advanced Concepts
1. Types & Debugging: Type checking, type variables, debugging techniques, and basic testing strategies.
2. Object-Oriented Programming: Inheritance, expression trees, abstract base classes, and properties for building well-structured class hierarchies.
3. Trees: Binary search trees (BSTs), AVL trees, and self-balancing tree insertion — including when and why to use each structure.
4. Graphs: Graph fundamentals, depth-first search (DFS), breadth-first search (BFS), and Dijkstra's shortest-path algorithm.
5. Functional Programming: Filter, map, lambda functions, and higher-order programming patterns in Python.
6. Heaps: Design, test, and implement min-heaps from scratch — covering initial design, test case creation, naive and optimized implementations, and black-box vs white-box testing.
7. Types with Type Variables: Parameterized types and generic programming.
Part 2: Software Development
1. Software Design Principles The software development process, software complexity, modular design, information hiding, general-purpose vs special-purpose design, and layered architecture.
2. Software Libraries: NumPy, Pandas (with real datasets including 311 service requests, ACA enrollment, and grades data), Click for command-line interfaces, and how to find and use external libraries.
3. Software Deployment: Software packaging, the client/server model, and web application basics.
4. Collaborative Development: Working in a team of four on a multi-milestone software project using Git, Linux, and VSCode in a real collaborative environment.

Final Grade: A
